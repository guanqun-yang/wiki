Welcome to Machine Learning Wiki!
=================================================
This is a knowledge base centered around machine learning. Its
primary purpose is to keep my learning and research organized but
you may also find it useful if we share similar academic background.

* To learn more about motivation of this wiki and notation used throughout
  the repository, please go to Prologue. (*recommended*)
* To find details about each topic, please go to each section. Note

  general and might be relatable to broader audience than other topics.
